# Memory-CS-GO-Aimbot
Script simples de aimbot para o Counter Strike: Global Offensive. 

Atenção: as offsets foram atualizadas para o presente momento (10/02); Ficará obsoleto na próximoa atualização tendo a necessidade de atualizar as mesmas. Utilize um dumper para obtê-las!
A tecla de ativação é F2; Aperte apenas no menu inicial e aguarde em torno de 15 segundos para iniciar uma sala.
